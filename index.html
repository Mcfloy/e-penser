E-penser

Automatic fansite communicating with Youtube API to show E-penser's videos.

View on GitHub Download .zip Download .tar.gz

Présentation d'e-penser

e-penser est une chaîne YouTube répondant à certaines questions aux travers de vidéos ludiques permettant de comprendre plus simplement un principe. Aussi Bruce Benamran, le créateur d'e-penser, est de facto propriétaire du contenu de ce fansite.

Présentation du fansite

Ce fansite a pour principe de lister de façons plus simples les vidéos qu'il propose.

Installation

Seule la copie du dossier "e-penser" sur votre site est nécessaire. Son fonctionnement ne repose que sur l'API YouTube. (Les fichiers README.md et LICENSE.md sont inutiles)
